Different kind of outputs

There are 4 different types of outputs in Javascript.


Alerts

The alerts create a pop up window that "alerts" the user of some important info i.e errors, browser not supported, etc.


Syntax


alert("Message to be displayed");




Prompts

Promts as it's name suggest, it asks the user for some kind of input data i.e. name, age, etc.

It's worth mentioning that for this to be useful throughout the whole page, we have to store the input value in a variable.


Syntax


prompt("Message to be requested");


Example

let name = prompt("What's your name");



Console

It displays the data in the browser's console included in the Developer's Tools section of the browser.


Syntax


console.log("Data to be displayed");




Document

It can write on the actual document the data.


Syntax


document.write("Data to be displayed");


Example


JAVASCRIPT Objects

Objects are entities with a collection of properties which describes the object and can be seen as an array


Syntax

let ObjectName= {

property name: "actual property"

}



Example

let camera= {

name: "Canon",

model: "350D",

price: "400 USD"

}



Access properties

To acces properties of an object we can do by using the dot(.) notation:


console.log(ObjectName.PropertyName)

console.log(camera.model)



We can also call using a variable, for example:

let p = "model";

console.log(camera[p]);

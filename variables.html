Variables

Definition

A variable is a container of data being that a number, a string, another variable or a null or undefined value.

A variable is declared using the keyword "VAR" or "LET" or "CONST", depending of the type of variable needed. Generally speaking the keyword "VAR" is being replaced by "LET" as VAR can be re-declared and is more of a function or global variable and "LET" is more contained within it's own scope or reach

A variable can be declared in two methods, by assignment or by initializing it

Examples of Variables

let a = 10;

let c = a+b;

const d = 50;

Examples of declaring a variable by assignment

let a;

a = 20;

Examples of declaring a variable by initializing it

let a = 10;

Things to consider or look out for



Undefined errors

This means that there is a variable being declared in our code, but it has no value assigned to it hence the undefined.

If by any chance we do a math calculation using it it would give out a "NaN" (Not a Number) error.



Variables are case sensitive

The following variables are not the same:

var someVariable;

var SomeVariable;

var SOMEVARIABLE;



Things to remember

Variables can be declared either by assigning the value in a separate line or declared by initializing it in the same line.

Variables declared using the VAR keyword can be re-declared as opposed to LET wich is more constarined to the scope reach

Variables that DO NOT change or likely to change throught the whole program is declared using the CONST keyword

CONST variables are generally named using all Caps
